Initial noise (z_T)
Random Gaussian latent, shape [64 channels], sampled in the VAE latent space. (Image-to-image instead starts from an encoded input image.) This is the latent the loop iteratively denoises.
Timestep + guidance
The current step's noise level t (decreasing T -> 0), plus a guidance scale. Embedded and fed to every block as AdaLN modulation.
Text conditioning
The prompt, encoded by CLIP + T5 into a sequence of width 4,096, conditions the denoiser (cross / joint attention). Computed once and reused every step.
DiT denoiser
The network applied at every step: a 19 MM-DiT dual-stream, 38 single-stream diffusion transformer that takes the current latent z_t (+ timestep + text conditioning) and predicts the noise to remove. Click to open its architecture.
Scheduler — FlowMatchEuler
FlowMatchEuler combines the predicted noise with z_t to produce z_{t-1}, one step toward a clean latent. Flow-matching (rectified-flow) sampler; 1,000 training timesteps; timestep shift 3.0. The loop repeats for N sampling steps (N chosen at inference, typically ~20-50 — it is not a config field).
VAE decoder
Once the loop reaches z_0 (clean latent), the VAE decoder maps it from latent space back to a full-resolution pixel image.
Output image
The generated image in pixel space.

Noisy latent (z_t)
VAE-space latent at timestep t; 64 channels. Conditioned on the prompt (encoded by CLIP + T5, width 4,096) and on a sinusoidal timestep + guidance embedding that drives AdaLN modulation across the stack. The denoiser predicts the noise/velocity to remove.
Patch embedding
Linear/conv patchify (patch 1x1) projecting latent patches to 3,072-d tokens; positional embedding added. Pooled text vector (768) joins the timestep conditioning.
Pre-block norm (shared)
LayerNorm; dim 3,072; shared input to both attention and FFN
Joint attention — single-stream
Full bidirectional attention over one concatenated image + text stream with shared Q/K/V; attention and MLP run in parallel on the same input (single-stream). Modulated by the timestep via AdaLN. Axial rotary position embedding (axes [16, 56, 56]).; input/output dim 3,072
Residual add (parallel)
layer input + attention output + FFN output (one combined step)
Feed-forward
FFN; GELU; hidden 12,288; input/output dim 3,072
Final modulation + projection
AdaLayerNorm-Out conditioned on the timestep vector, then a linear projection from 3,072-d tokens back to latent patches.
Unpatchify -> predicted noise
Reassemble predicted patches into the latent grid: the denoiser's output for this step — the noise (or velocity) eps(z_t, t) to remove. The scheduler (in the sampling loop) uses it to step z_t -> z_{t-1}.
Query projection
Linear; 3,072 -> 3,072 (24 heads x 128 dims)
Key projection
Linear; 3,072 -> 3,072 (24 KV-heads x 128 dims). Cache ports show K/V write/read during generation: arrowhead for write, blunt tail for read.
Value projection
Linear; 3,072 -> 3,072 (24 KV-heads x 128 dims). Cache ports show K/V write/read during generation: arrowhead for write, blunt tail for read.
Scaled attention scores
Per head: QK^T / sqrt(dim); dot-product scores scaled for numerical stability
Softmax weights
Normalize each query row into attention weights over source tokens
Apply values
Multiply attention weights by V to produce one context vector per head
Concatenate heads
Stack all 24 per-head context vectors back into width 3,072
Output projection
Linear; 3,072 -> 3,072 (mixes information across heads)
Input projection
Linear; 3,072 -> 12,288
GELU activation
Element-wise non-linearity applied after the input projection
Output projection
Linear; 12,288 -> 3,072
