Initial noise
z_T: random Gaussian latent, shape [4 channels], sampled in the VAE latent space. (Image-to-image instead starts from an encoded input image.) This is the latent the loop iteratively denoises.
Timestep
The current step's noise level t (decreasing T -> 0). Embedded and fed to every block as AdaLN modulation.
Text prompt
The conditioning prompt, encoded by 1 text encoder (CLIP).
CLIP text encoder
CLIP: produces a pooled prompt vector, used as global conditioning (AdaLN modulation). Frozen; run once and reused every sampling step.
U-Net denoiser
The network applied at every step: a 4-stage convolutional U-net (down → mid → up with skip connections) that takes the latent z_t (+ timestep + text conditioning) and predicts the noise. Click to open its architecture.
Scheduler — Euler
Euler combines the predicted noise with z_t to produce z_{t-1}, one step toward a clean latent. 1,000 training timesteps. The loop repeats for N sampling steps (N chosen at inference, typically ~20-50 — it is not a config field).
VAE decoder
Once the loop reaches z_0 (clean latent), the VAE decoder maps it from latent space back to a full-resolution pixel image.
Output image
The generated image in pixel space.
